Example for Failure Criteria ❌

Failure Example 1: Input Field Missing Programmatic Label

Scenario: The input field has a placeholder but no visible or programmatic label.

**Compliance Justification:** While the placeholder provides a visual cue, it is **not a sufficient programmatic label**. A screen reader will only announce the input as "edit text" or similar, leaving the user unsure of its purpose. The `for`/`id` association or a proper ARIA label is missing. This fails SC 3.3.2.
